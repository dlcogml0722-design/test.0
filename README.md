# ⛩️ 나만의 성지순례 (K-Webtoon & Web Novel Pilgrimage Map)

K-웹툰 및 웹소설의 실제 배경지 또는 촬영 장소를 지도 상에서 간편하게 탐색하고, 나만의 순례 경로를 저장하고, 길찾기를 제공하는 글래스모피즘 스타일의 다국어 지도 서비스입니다.

---

## 🚀 빠른 시작 방법 (사용자 가이드)

### 1. 로컬 개발 서버 실행하기
본 서비스는 별도의 백엔드 데이터베이스 설치 없이 웹 브라우저에서 바로 동작하는 **단일 페이지 애플리케이션(SPA)**입니다. 로컬 환경에서 지도가 정상적으로 표시되도록 웹 서버를 통해 실행합니다.

터미널이나 명령 프롬프트(CMD/PowerShell)를 열고 프로젝트 폴더로 이동한 뒤, 아래 명령어를 실행하십시오:

```bash
# Node.js가 설치되어 있는 경우 (추천)
npx serve . -l 3000

# 파이썬이 설치되어 있는 경우
python -m http.server 3000
```

서버가 켜지면 웹 브라우저를 열고 다음 주소로 접속합니다:
👉 **[http://localhost:3000](http://localhost:3000)**

## 📱 주요 기능 및 사용법 안내

### 1. 🗺️ 다크 테마 지도 및 실제 도로 길찾기 라인
- **실제 도로망 라인 드로잉 (OSRM 연동)**: 단순히 출발지와 장소를 잇는 직선이 아니라, 실제 운전/도보 도로망의 구불구불한 꺾임선을 따라 **실시간으로 실제 지도 경로선(Polyline)**을 덧그려 보여줍니다.
- **다크 테마**: CartoDB Dark Matter 타일로 프리미엄 다크 지도.
- **커스텀 마커**: 작품별 이모지와 실제 공식 표지가 들어간 마커 제공.

### 2. 📍 현위치 추적 (GPS) 및 정렬
- **현위치 버튼**: 지도 화면 우측 상단 줌 컨트롤 바로 밑에 `📍` 버튼이 고정 배치되어 있습니다.
- **서울 기준 예외 처리**: 사용자가 서울이 아닌 지역(예: 부산 등)에 있거나 GPS 권한이 꺼져 있어도 서울을 기준으로 모든 스팟과 길찾기 경로를 안전하게 표현해 줍니다.

### 3. 🖼️ AI 배제 정품 웹툰/포스터 썸네일
- AI가 생성한 인위적인 이미지 대신, **네이버웹툰/위키미디어/TMDb에 기등재된 실제 공식 정품 웹툰 단행본 표지 및 포스터**를 온라인 다이렉트 연결하여 100% 엑박(Broken Image) 없이 생생한 실제 이미지를 감상할 수 있습니다.

### 4. 🚶 도보/대중교통 길찾기 가이드 및 딥링크
- **대중교통 상세 루트**: 상세 모달창 하단에 나무위키 고증을 거친 실제 버스/지하철 탑승 안내와 환승 방법이 상세하게 나열됩니다.
- **외부 맵 딥링크**: 하단 버튼을 클릭하면 네이버 지도 또는 카카오맵 서비스로 바로 연결되어 실제 실시간 도보/대중교통 내비게이션 경로를 열어볼 수 있습니다.

### 5. ❤️ 개인 저장 (스크랩)
- 각 스팟 카드의 하트(`🤍` -> `❤️`) 버튼을 누르면 개인 브라우저(`localStorage`)에 보관되어 영구히 저장됩니다.
- 하단 네비게이션 바의 **[스크랩]** 탭을 누르면 내가 찜해 둔 장소들만 모아서 보거나 지도에 띄울 수 있습니다.

## 🎨 기술 명세서
- **Core**: Vanilla HTML5, Javascript (ES6+)
- **Map Library**: Leaflet.js (OpenStreetMap 기반)
- **Map Tiles**: CartoDB Dark Matter (API Key 필요 없음)
- **Styling**: Vanilla CSS3 (CSS Variables, Flexbox, CSS Grid)
- **i18n**: 한국어, 영어, 일본어 3개국어 완벽 대응
- **Responsive**: 모바일 브라우저 표준 뷰포트 최적화 (Safe Area 대응)
